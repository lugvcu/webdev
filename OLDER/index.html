Welcome to the GNU & Linux User Group of VCU

We are a student group at VCU, dedicated to promoting free software and open source applications such as GNU/Linux. What is Linux?

Linux is an operating system just like Windows. It is the main software on a computer that allows other programs and systems to run. At this point Linux is  a fully developed, user-friendly operating system, much like Mac OSX and Windows. Linux is freely available, from our download page as well as from many sites on the Internet.

Advantages:
Linux and the vast majority of software that runs on it is open-source and free of charge. Programs can be easily added and modified both freely and legally.
GNU/Linux is also community maintained, so critical bugs and security issues are usually handled very quickly.
There are almost no viruses for GNU/Linux, and most people do not use any sort of virus protection software, although it is available.

Disadvantages:
While Linux is growing in popularity every day, Mac OSX and Windows are still the focus of hardware and software development companies. This means that the software that runs on the other systems probably wont run (natively) on Linux. There are exceptions however. Please feel free to ask the group to explain this.
While most hardware is automatically supported by current distributions, there are exceptions that need special attention. This is another service that the VCU Linux User Group is happy to help with.
