Welcome to our Website!



Who are the LUG?

We are a student group at VCU, dedicated to promoting free software and open source applications such as GNU/Linux. We enjoy helping students and the VCU community better understand the benefits of free and open software. The group is also dedicated to introducing new people to these programs and helping them begin using Linux and other open source projects.

Our goal is to provide a friendly source of information and help regarding all things open source to the VCU community. The Free Software Foundation developed, among other significant documents, the following definition which you may find helpful in understanding this organization.

The best way to keep up to date on group events is to join the email discussion forum. Please use the form on our Connect page to contact the officers or for any website related comments.



What is Linux?

Linux is an operating system just like Windows. It is the main software on a computer that allows other programs and systems to run. At this point Linux is a fully developed, user-friendly operating system, much like Mac OSX and Windows. Linux is freely available, from our download page as well as from many sites on the Internet.

Advantages:

Linux and the vast majority of software that runs on it is open-source and free of charge. Programs can be easily added and modified both freely and legally. GNU/Linux is also community maintained, so critical bugs and security issues are usually handled very quickly. There are almost no viruses for GNU/Linux, and most people do not use any sort of virus protection software, although it is available.

Selected Linux Live Discs

If you want to try Linux, but don't know where to start, try one of the following Linux Live Discs. A Linux live disc is usually burned to a writable CD or DVD and allows one to boot linux onto a computer without installing it. On all of these, one can also install that distribution onto the computer as well. Recently purchased computers all have 64-bit cpus, except Netbooks. If you are unsure which Live Disc to try choose either Fedora Gnome 32-bit or Ubuntu 32-bit. On computer which already have an operating system installed (typically windows) a new user should seek help if they wish to dual boot between the two operating systems.

Membership

Any student interested in joining the group can become a member. We have also allowed honorary membership to non-VCU students in the past.
